- 支持批量采集，多个产品同时进行
- 自动存储和管理，随时查看历史数据

## 📋 操作步骤

### 准备工作

#### 前置条件

- ✅ 已安装 Chrome 浏览器（版本 90+）
- ✅ 已注册 VOC-Master 账号
- ✅ 已安装 VOC-Master Chrome 插件
- ✅ 已登录 Amazon 账号（任意站点）

#### 环境检查

1. **检查插件状态**
   - 点击浏览器工具栏的 VOC-Master 图标
   - 确认显示"已登录"
   - 确认"后端服务正常"（绿色✅）

2. **检查 Amazon 登录状态**
   - 访问 Amazon.com（或其他站点）
   - 确认右上角显示"Hello, [您的名字]"
   - 如果未登录，请先登录

**💡 提示：** 如果长时间未使用 Amazon，可能需要重新登录。

### 详细操作

#### Step 1：选择要采集的产品

**方式 1：搜索产品（推荐）**

1. 打开 Amazon.com

2. 在搜索框输入关键词
   - 示例："wireless bluetooth earbuds"

3. 点击搜索结果中的任意产品

4. 进入产品详情页

**方式 2：直接访问产品链接**

1. 如果您已有产品链接（如竞品链接）

2. 直接在浏览器中打开
   - 示例：`https://www.amazon.com/dp/B08N5WRWNW`

3. 确保页面加载完成，能看到评论区

**💡 选品建议：**

| 评论数量 | 采集时间 | 分析效果 | 建议 |
|---------|---------|---------|------|
| < 20 条 | 1 分钟 | ❌ 样本不足，分析不全面 | 不推荐 |
| 20-100 条 | 2-5 分钟 | ✅ 快速体验，基本分析 | 适合快速测试 |
| 100-500 条 | 5-15 分钟 | ✅ 数据充足，分析准确 | **推荐** |
| 500-1000 条 | 15-30 分钟 | ✅ 数据丰富，深度分析 | 适合重点产品 |
| > 1000 条 | 30+ 分钟 | ✅ 大数据分析 | 适合热门产品 |

#### Step 2：打开采集面板

1. 在产品详情页，点击浏览器工具栏的 **VOC-Master 图标**

2. 在弹出窗口中，点击 **"打开采集面板"**

3. 采集面板从页面右侧滑出

4. 确认产品信息自动识别
   ```
   ✅ 产品标题：Wireless Bluetooth Earbuds...
   ✅ ASIN：B08N5WRWNW
   ✅ 当前站点：Amazon.com
   ✅ 评论总数：1,234 条
   ```

**💡 提示：** 如果产品信息未自动识别，刷新页面后重试。

#### Step 3：配置采集参数

**参数说明：**

##### 1. 采集星级

```
☑️ 5星 (推荐：勾选)
☑️ 4星 (推荐：勾选)
☑️ 3星 (推荐：勾选)
☑️ 2星 (推荐：勾选)
☑️ 1星 (推荐：勾选)
```

**推荐配置：** 全选（了解完整口碑）

**特殊场景：**
- 只看好评：仅勾选 4-5 星（挖掘卖点）
- 只看差评：仅勾选 1-2 星（发现问题）
- 只看中性评价：仅勾选 3 星（了解改进空间）

##### 2. 采集数量

```
[滑块] 10 ←──────●───→ 1000
       当前：200 条
```

**推荐配置：**
- 首次体验：100-200 条
- 日常分析：200-500 条
- 深度分析：500-1000 条

**💡 提示：** 如果评论总数少于设置值，会采集全部评论。

##### 3. 采集速度

```
○ 标准（1.5-2 秒/条，推荐）
○ 快速（0.8-1 秒/条）
○ 极速（0.3-0.5 秒/条，风险高）
```

**推荐配置：** 标准（稳定性最好）

**速度对比：**

| 模式 | 速度 | 采集 500 条耗时 | 风险 | 适用场景 |
|------|------|---------------|------|---------|
| **标准** | 1.5-2 秒/条 | 12-17 分钟 | ✅ 低 | 日常使用（推荐） |
| **快速** | 0.8-1 秒/条 | 7-8 分钟 | ⚠️ 中 | 紧急任务 |
| **极速** | 0.3-0.5 秒/条 | 2-4 分钟 | ⛔ 高 | 仅限测试 |

**⚠️ 风险说明：**
- "标准"模式：几乎不会触发 Amazon 反爬机制
- "快速"模式：偶尔可能被限制，需暂停 5-10 分钟
- "极速"模式：容易被限制，可能导致采集失败，仅用于测试

##### 4. 翻译模式（工作流模式）

```
○ 一步到位（采集+翻译+分析，推荐）
○ 只翻译（采集+翻译，等待手动分析）
```

**模式对比：**

| 模式 | 流程 | 适用场景 |
|------|------|---------|
| **一步到位** | 采集 → 翻译 → 洞察提取 → 主题提取 → 报告生成（全自动） | 快速获取完整分析结果（推荐） |
| **只翻译** | 采集 → 翻译 → 停止（需手动点击"开始分析"） | 需要先查看翻译结果或手动编辑维度 |

**💡 推荐：** 首次使用选择"一步到位"，体验完整流程。

#### Step 4：开始采集

1. 确认所有参数配置完成

2. 点击 **"开始采集"** 按钮

3. 采集开始，实时显示进度
   ```
   ⏳ 正在采集... 
   
   进度：123 / 200 (61.5%)
   预计剩余时间：2 分 18 秒
   
   当前状态：正在加载评论...
   ```

4. 采集过程中可以：
   - **暂停**：点击"暂停"按钮，稍后点击"继续"
   - **停止**：点击"停止"按钮，结束采集（已采集的评论已保存）
   - **调整速度**：暂停后可以调整采集速度

5. 采集完成提示
   ```
   ✅ 采集完成！
   
   已采集：198 条评论
   已提交：198 条评论
   翻译中：198 条评论（预计 3-5 分钟）
   
   [前往控制台查看分析]
   ```

**💡 提示：** 采集完成后，翻译在后台异步进行，您可以立即关闭页面。

#### Step 5：查看采集结果

**方式 1：从采集面板跳转**

1. 采集完成后，点击 **"前往控制台查看分析"**

2. 浏览器自动打开 VOC-Master 控制台

3. 您会看到产品列表中出现了刚才采集的产品

**方式 2：从控制台查找**

1. 打开 VOC-Master 控制台首页

2. 在产品列表中找到刚才采集的产品
   - 产品图片
   - 产品标题
   - 采集时间："刚刚" 或 "2 分钟前"

3. 点击产品卡片，进入产品详情页

#### Step 6：查看翻译结果

1. 进入产品详情页

2. 默认显示 **"评论列表"** 标签页

3. 翻译进度显示
   ```
   ⏳ 翻译中... 167 / 198 (84.3%)
   预计剩余时间：1 分钟
   ```

4. 翻译完成后，显示中英双语评论
   ```
   ⭐⭐⭐⭐⭐  John D.  2024-01-15
   
   [英文原文]
   This is amazing! Best purchase ever. Battery lasts forever 
   and the sound quality is top-notch. Highly recommend!
   
   [中文翻译]
   太棒了！买过最好的东西。电池续航超长，音质一流。
   强烈推荐！
   
   👍 254 人认为有帮助
   ```

5. 功能体验
   - **筛选**：按星级筛选（1-5 星）
   - **排序**：按时间/有用性排序
   - **搜索**：关键词搜索（支持中英文）
   - **导出**：导出为 Excel 或 CSV

**💡 提示：** 如果翻译还在进行中，可以先查看已翻译的评论，也可以等待全部完成。

### 批量采集技巧

#### 同时采集多个产品

1. **打开多个浏览器标签页**
   - 每个标签页打开一个产品详情页

2. **逐个打开采集面板**
   - 在每个标签页点击插件图标
   - 打开采集面板，配置参数

3. **逐个开始采集**
   - 回到第一个标签页，点击"开始采集"
   - 切换到第二个标签页，点击"开始采集"
   - 依次类推

4. **后台自动处理**
   - 评论自动提交到服务器
   - 后台异步翻译和分析
   - 您可以关闭浏览器，稍后查看结果

**💡 建议：** 同时采集不超过 5 个产品，避免浏览器卡顿和 Amazon 限制。

#### 竞品批量采集

**场景：** 对比分析 5 个竞品

1. **准备竞品链接**
   - 在 Excel 或记事本中整理竞品 ASIN 或链接
   - 示例：
     ```
     竞品A：https://www.amazon.com/dp/B08N5WRWNW
     竞品B：https://www.amazon.com/dp/B08N5XXYYZ
     竞品C：https://www.amazon.com/dp/B08N5AABBCC
     ```

2. **逐个打开并采集**
   - 复制第一个链接，粘贴到浏览器打开
   - 打开采集面板，配置参数，开始采集
   - 重复操作，完成全部 5 个竞品

3. **查看对比分析**
   - 回到控制台，点击"竞品对比"
   - 选择这 5 个产品，生成对比报告

**时间估算：** 5 个竞品，每个 200 条评论，总耗时约 30-40 分钟（包括采集和翻译）。

## 💼 实战案例

### 案例 1：新品上市监控

**背景：**
- 某卖家在 Amazon 上线了一款蓝牙耳机
- 上线 2 周，已有 45 条评论
- 希望快速了解用户反馈，发现问题

**操作步骤：**

1. 访问自家产品页面

2. 打开采集面板，配置参数
   - 采集星级：全选
   - 采集数量：全部（45 条）
   - 采集速度：标准
   - 翻译模式：一步到位

3. 开始采集，2 分钟完成

4. 查看翻译结果，发现问题
   - 发现 8 条评论提到"充电盒盖松动"
   - 发现 5 条评论提到"说明书不清晰"

5. 立即采取行动
   - 联系工厂改进充电盒设计
   - 制作图文并茂的电子说明书

**结果：**
- 及时发现质量问题，避免更多差评
- 2 周内完成产品改进，新批次产品好评率提升 18%

### 案例 2：竞品分析找差异化

**背景：**
- 某卖家准备进入智能手表市场
- 市场竞争激烈，不知如何差异化
- 决定先分析 3 个头部竞品

**操作步骤：**

1. 选择 3 个竞品（Best Seller 排行榜前 3）

2. 逐个采集评论
   - 竞品 A：采集 300 条
   - 竞品 B：采集 300 条
   - 竞品 C：采集 300 条

3. 查看翻译结果，发现共性痛点
   - 所有竞品都有"电池续航短"的差评（高频）
   - 所有竞品都有"表带易损坏"的差评（中频）

4. 查看卖点分布
   - 竞品 A：外观精美（高频好评）
   - 竞品 B：功能丰富（高频好评）
   - 竞品 C：价格实惠（高频好评）

5. 制定差异化策略
   - **核心差异点**：主打"超长续航"（7 天 vs 竞品 3 天）
   - **辅助差异点**：金属表带（解决易损坏问题）
   - **定位**：中高端市场（避开价格战）

**结果：**
- 产品上线后，"超长续航"成为核心卖点，Listing 转化率 4.2%（行业平均 2.8%）
- 避开红海竞争，找到差异化切入点

**背景：**

- 决定从用户评论中挖掘卖点

**操作步骤：**

1. 采集自家产品评论（280 条）

2. 查看翻译结果，统计高频好评
   - "画质清晰"：42 次提及
   - "夜视效果好"：38 次提及
   - "双向语音很实用"：35 次提及
   - "安装简单"：28 次提及

3. 对比现有 Listing
   - **现有标题**：Baby Monitor with Camera
   - **现有五点描述**：
     - HD camera
     - Night vision
     - Two-way audio
     - Easy setup

## ⚠️ 注意事项

### 1. 采集频率控制

**❌ 不推荐：**
- 短时间内多次采集同一产品（如 1 小时内采集 3 次）
- 同时采集超过 10 个产品
- 连续采集超过 2 小时不休息

**✅ 推荐：**
- 同一产品每天最多采集 1-2 次
- 同时采集不超过 5 个产品
- 每采集 30 分钟，休息 5-10 分钟

**原因：** 频繁采集可能触发 Amazon 反爬机制，导致账号被限制。

### 2. Amazon 账号安全

**保护您的 Amazon 账号：**

- ✅ 使用主账号采集（不要用小号）
- ✅ 采集速度选择"标准"模式
- ✅ 遵守采集频率建议
- ✅ 不要在采集过程中频繁切换 IP

**⚠️ 如果遇到限制：**

1. 停止采集，等待 30-60 分钟
2. 重新登录 Amazon 账号
3. 降低采集速度，减少采集数量
4. 如果仍然无法采集，联系客服

**💡 提示：** VOC-Master 插件设计时已充分考虑安全性，正常使用不会导致账号问题。

### 3. 翻译质量

**AI 翻译的局限性：**

- ✅ 大部分评论翻译准确率 > 95%
- ⚠️ 极少数专业术语或俚语可能翻译不准确
- ⚠️ 特殊符号、表情符号可能丢失

**如何提高翻译质量：**

1. **查看原文对照**
   - 如果翻译看起来奇怪，查看英文原文
   - 右键点击翻译，选择"显示原文"

2. **手动修正**
   - 点击翻译右上角的"编辑"图标
   - 修改翻译内容
   - 点击"保存"

3. **反馈问题**
   - 点击翻译右上角的"反馈"图标
   - 提交翻译问题
   - 帮助我们持续优化

### 4. 数据隐私

**VOC-Master 的数据保护承诺：**

- 🔒 所有数据传输采用 HTTPS 加密
- 🔒 评论数据仅用于分析，不会分享给第三方
- 🔒 支持私有服务器部署，数据不出公司
- 🔒 您可以随时删除采集的产品和评论

## ❓ 常见问题

### Q1：采集速度慢，如何提速？

**A：** 尝试以下方法：
1. 切换到"快速"或"极速"模式（注意风险）
2. 减少采集数量（如从 500 减少到 200）
3. 检查网络连接，确保网速稳定
4. 关闭其他占用网络的程序

### Q2：采集中断了怎么办？

**A：** 不用担心，VOC-Master 支持断点续采：
1. 重新打开采集面板
2. 系统会自动检测已采集的评论
3. 点击"继续采集"，从断点处继续
4. 如果无法继续，点击"重新采集"

### Q3：翻译速度慢，如何加速？

**A：** 翻译在后台异步进行，速度取决于以下因素：
- **评论数量**：100 条约 2 分钟，500 条约 10 分钟
- **服务器负载**：高峰期可能稍慢
- **网络状况**：网络不稳定会影响速度

**加速建议：**
- 错峰使用（避开高峰期：10:00-12:00, 14:00-16:00）
- 批量翻译自动启用，无需手动干预

### Q4：翻译质量不满意怎么办？

**A：** 您可以：
1. **手动修正**：点击翻译右上角的"编辑"图标，修改内容
2. **查看原文**：右键点击翻译，选择"显示原文"
3. **反馈问题**：点击"反馈"图标，提交翻译问题

我们会持续优化翻译模型，感谢您的反馈！

### Q5：可以采集其他平台的评论吗（如 eBay、Walmart）？

**A：** 目前 VOC-Master 仅支持 Amazon 平台。我们计划在未来版本支持更多平台，敬请期待。

### Q6：采集的评论会保存多久？

**A：** 永久保存，除非您手动删除。您可以随时回到产品详情页查看历史数据。

### Q7：如何删除不需要的产品？

**A：** 在产品列表或详情页，点击右上角的"删除"按钮，确认删除即可。删除后无法恢复，请谨慎操作。

### Q8：采集的评论可以导出吗？

**A：** 可以。进入产品详情页，点击"导出"按钮，选择格式（Excel 或 CSV），即可下载。

### Q9：翻译结果可以批量导出吗？

**A：** 可以。在产品列表页，勾选多个产品，点击"批量导出"，选择"翻译结果"，即可下载。

### Q10：如何查看采集历史记录？

**A：** 在控制台首页，点击左侧菜单的"采集历史"，可以查看所有采集记录和状态。
