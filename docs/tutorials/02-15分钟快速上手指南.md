# 15分钟快速上手指南

## 📖 指南简介

本指南将带您在 **15 分钟内**完成从安装到分析的完整体验，快速掌握 VOC-Master 的核心功能。

**学习目标：**
- ✅ 安装并配置 Chrome 插件
- ✅ 完成第一次评论采集
- ✅ 查看翻译和分析结果
- ✅ 生成第一份智能报告

**适用人群：** 第一次使用 VOC-Master 的新用户

## ⏱️ 时间分配

| 步骤 | 内容 | 预计时间 |
|------|------|---------|
| **环境准备** | 注册账号、安装插件 | 3 分钟 |
| **第一次采集** | 选择产品、配置采集 | 5 分钟 |
| **查看分析结果** | 查看翻译、洞察、画像 | 5 分钟 |
| **生成报告** | 一键生成智能报告 | 2 分钟 |

---

## 第一部分：环境准备（3 分钟）

### Step 1.1：注册账号

1. 访问 VOC-Master 控制台
   - 打开浏览器，访问：`https://你的域名.com`
   
2. 点击右上角 **"注册"** 按钮

3. 填写注册信息
   - 邮箱地址
   - 设置密码（建议：8位以上，包含字母和数字）
   - 邮箱验证码

4. 点击 **"完成注册"**

5. 登录系统
   - 输入邮箱和密码
   - 点击 **"登录"**

### Step 1.2：安装 Chrome 插件

1. 打开 Chrome 浏览器

2. 访问插件安装页面
   - **方式 1（推荐）**：从 VOC-Master 控制台首页点击 **"下载插件"**
   - **方式 2**：Chrome 应用商店搜索 "VOC-Master"

3. 点击 **"添加至 Chrome"**

4. 在弹出窗口点击 **"添加扩展程序"**

5. 确认安装成功
   - 浏览器工具栏出现 VOC-Master 图标 🎯
   - 点击图标，看到 "未登录" 提示

### Step 1.3：插件登录配置

1. 点击浏览器工具栏的 VOC-Master 图标

2. 在弹出窗口点击 **"前往登录"**

3. 输入您的邮箱和密码

4. 登录成功后，插件状态变为 **"已登录"**

5. 确认后端连接
   - 插件底部显示：✅ 后端服务正常
   - 显示您的用户名

**环境准备完成！** ✅

---

## 第二部分：第一次采集（5 分钟）

### Step 2.1：选择一个 Amazon 产品

1. 打开新标签页，访问 **Amazon.com**（或其他站点，如 Amazon.co.uk）

2. 搜索一个您感兴趣的产品
   - **建议：** 选择评论数在 50-500 条的产品（采集速度快，数据量适中）
   - **示例：** 搜索 "wireless mouse" 或 "coffee maker"

3. 进入产品详情页
   - 点击搜索结果中的任意产品
   - 确保页面加载完成，能看到评论区

**💡 提示：** 评论数太少（<20 条）分析结果不够全面，太多（>1000 条）采集时间较长，初次体验建议 50-200 条。

### Step 2.2：打开采集面板

1. 在产品详情页，点击浏览器工具栏的 VOC-Master 图标

2. 在弹出窗口中，点击 **"打开采集面板"**

3. 采集面板会在页面右侧滑出

4. 确认产品信息
   - 产品标题自动识别
   - 产品 ASIN 自动识别
   - 当前评论总数显示

### Step 2.3：配置采集参数

**默认配置（推荐新手）：**

```
✅ 采集星级：1-5 星全选（勾选）
✅ 采集数量：100 条（滑块调整）
✅ 采集速度：标准（推荐）
✅ 翻译模式：自动翻译（一步到位）
```

**参数说明：**

| 参数 | 说明 | 推荐设置 |
|------|------|---------|
| **采集星级** | 选择采集哪些星级的评论 | 全选（了解完整口碑） |
| **采集数量** | 最多采集多少条评论 | 100-200 条（首次体验） |
| **采集速度** | 标准/快速/极速 | 标准（稳定性好） |
| **翻译模式** | 一步到位/先采集后翻译 | 一步到位（自动化） |

**💡 提示：** 首次体验建议采集 100 条左右，速度选择"标准"，避免触发 Amazon 反爬机制。

### Step 2.4：开始采集

1. 确认配置无误后，点击 **"开始采集"** 按钮

2. 采集进度实时显示
   ```
   正在采集... 23 / 100 (23%)
   预计剩余时间：2 分 30 秒
   ```

3. 采集过程中您可以：
   - 查看实时进度条
   - 看到已采集的评论数量
   - 随时点击 **"暂停"** 或 **"停止"**

4. 采集完成提示
   ```
   ✅ 采集完成！
   已采集：98 条评论
   已提交：98 条评论
   翻译中：98 条评论
   ```

5. 点击 **"前往控制台查看分析"**

**采集完成！** ✅

**常见问题：**

- **Q：采集速度很慢？**  
  A：选择"快速"或"极速"模式，但注意不要频繁采集同一产品。

- **Q：采集中断了？**  
  A：没关系，已采集的评论已保存。重新打开采集面板，点击"继续采集"。

- **Q：提示"请先登录 Amazon"？**  
  A：确保您已登录 Amazon 账号，刷新页面后重试。

## 第三部分：查看分析结果（5 分钟）

### Step 3.1：进入产品详情页

1. 采集完成后，点击 **"前往控制台查看分析"**

2. 浏览器自动跳转到 VOC-Master 控制台

3. 您会看到产品列表中出现了刚才采集的产品
   - 产品图片
   - 产品标题
   - 评论总数
   - 翻译进度（可能显示"翻译中...98%"）

4. 点击产品卡片，进入产品详情页

**💡 提示：** 如果翻译还在进行中（显示进度条），等待 1-2 分钟后刷新页面。

### Step 3.2：查看翻译结果

1. 进入产品详情页后，默认看到 **"评论列表"** 标签页

2. 评论显示格式
   ```
   ⭐⭐⭐⭐⭐  John D.  2024-01-15
   
   [英文原文]
   This is amazing! Best purchase ever. Battery lasts forever.
   
   [中文翻译]
   太棒了！买过最好的东西。电池续航超长。
   
   👍 254 人认为有帮助
   ```

3. 功能体验
   - **筛选**：按星级筛选（1-5 星）
   - **排序**：按时间/有用性排序
   - **搜索**：关键词搜索（支持中英文）
   - **导出**：导出为 Excel 或 CSV

**💡 提示：** 对比翻译质量，感受"拒绝翻译腔"的自然表达。

### Step 3.3：查看深度洞察

1. 点击页面顶部 **"洞察分析"** 标签页

2. 您会看到 5 类洞察自动归类
   ```
   📊 洞察总览
   优势卖点：32 条
   痛点问题：18 条
   用户建议：12 条
   使用场景：8 条
   情绪反馈：15 条
   ```

3. 点击 **"优势卖点"** 查看详情
   - 看到洞察按维度分组（如：电池续航、外观设计、使用体验）
   - 每个维度下方显示具体洞察和频次
   - 点击洞察可查看原始评论证据

4. 体验证据溯源
   - 点击任意洞察条目
   - 右侧弹出 **"证据面板"**
   - 显示该洞察对应的原始评论引用

**示例：**

```
维度：电池续航
├─ 续航时间长 (12次提及)
│  └─ 证据：
│      "Battery lasts forever" - ⭐⭐⭐⭐⭐ John D.
│      "Amazing battery life" - ⭐⭐⭐⭐⭐ Sarah M.
│
├─ 充电速度快 (8次提及)
│  └─ 证据：...
```

**💡 提示：** 每个洞察都有数据支撑，点击可查看原始评论，确保结果可信。

## 第四部分：生成报告（2 分钟）

### Step 4.1：选择报告类型

1. 在产品详情页，点击页面顶部 **"智能报告"** 标签页

2. 如果还没有报告，点击 **"生成报告"** 按钮

3. 选择报告类型（4 种）

**报告类型说明：**

| 报告类型 | 适用角色 | 核心内容 |
|---------|---------|---------|
| **综合战略报告** | CEO/高管 | SWOT 分析、PMF 评估、优先行动项 |
| **运营市场报告** | CMO/运营 | 核心卖点、目标人群、Listing 优化 |
| **产品迭代报告** | CPO/产品 | 致命 Bug、用户需求、质量评分 |
| **供应链报告** | 供应链/QC | 质量问题、QC 清单、退货原因 |

4. 首次体验建议选择 **"运营市场报告"**（最实用）

5. 点击 **"生成报告"**

### Step 4.2：等待生成（约 30 秒）

1. 生成过程中显示进度

## ❓ 常见问题

### Q1：翻译需要多长时间？

**A：** 通常 100 条评论需要 2-3 分钟，500 条评论需要 10-15 分钟。翻译是后台异步进行，您可以随时查看进度。

### Q2：洞察提取和用户画像什么时候生成？

**A：** 默认采集完成后自动触发。如果选择"先采集后翻译"模式，需要在产品详情页手动点击"开始分析"。

### Q3：报告可以重新生成吗？

**A：** 可以。点击报告右上角的"重新生成"按钮，可以选择不同的报告类型。每个产品每小时可重新生成 1 次。

### Q4：采集的评论会保存多久？

**A：** 永久保存，除非您手动删除产品。您可以随时回来查看历史分析结果。

### Q5：可以同时采集多个产品吗？

**A：** 可以。每个产品对应一个浏览器标签页，可以同时打开多个采集面板。建议同时采集不超过 3 个产品，避免浏览器卡顿。

### Q6：如何邀请团队成员一起使用？

**A：** 在控制台右上角点击"团队管理"，可以邀请团队成员并设置权限。详见 [高级技巧：团队协作](./09-高级技巧与最佳实践.md#团队协作)。
