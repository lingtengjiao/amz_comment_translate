# 市场洞察功能 - 快速测试指南

## 🚀 快速开始

### 1. 后端API测试（推荐先运行）

```bash
# 方式1: 使用快速测试脚本
./scripts/quick_test_market_insight.sh

# 方式2: 直接运行Python测试
python3 scripts/test_market_insight_api.py
```

**预期结果**: 所有测试通过 ✅

### 2. 启动服务进行完整测试

#### 启动后端

```bash
cd backend
uvicorn app.main:app --reload
```

后端将在 `http://localhost:8000` 启动

#### 启动前端

```bash
cd frontend
npm run dev
```

前端将在 `http://localhost:5173` 启动（或查看终端输出）

### 3. 前端功能测试

1. **打开浏览器**访问前端地址
2. **选择产品**（至少2个，最多10个用于市场洞察）
3. **点击"开始分析"**
4. **选择分析类型**：
   - 对比分析（2-5个产品）
   - 市场洞察（2-10个产品）
5. **输入项目名称**并提交
6. **等待分析完成**（1-2分钟）
7. **查看结果页面**
