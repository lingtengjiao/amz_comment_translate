VOC-Master 隐私政策

Privacy Policy

最后更新日期 / Last Updated: 2026年1月16日

重要提示：本隐私政策说明了 VOC-Master Chrome 扩展（以下简称"本扩展"或"我们"）如何收集、使用、存储和保护您的个人信息。使用本扩展即表示您同意本隐私政策的条款。

1. 信息收集 / Information Collection

1.1 我们收集的信息类型

本扩展在您使用过程中会收集以下类型的信息：

1.1.1 用户认证信息

  • 认证令牌（Token）：用于维持您的登录状态
  • 用户账户信息：包括邮箱地址、用户名等（仅在您登录时收集）
  • 会话信息：Token 过期时间、签发时间等

存储位置：这些信息仅存储在您的浏览器本地（chrome.storage.local），不会上传到我们的服务器。

1.1.2 Amazon 产品信息

当您使用本扩展采集评论时，我们会收集以下产品相关信息：

  • 产品 ASIN（Amazon 标准识别号）
  • 产品标题、图片 URL
  • 产品价格、平均评分
  • 产品类目信息
  • 产品五点卖点（Bullet Points）

1.1.3 Amazon 评论数据

本扩展的核心功能是采集 Amazon 产品评论，收集的数据包括：

  • 评论 ID（Amazon 内部标识符）
  • 评论内容（标题、正文）
  • 评分（1-5 星）
  • 评论作者名称（公开信息）
  • 评论发布日期
  • 是否认证购买
  • 有用票数
  • 评论链接
  • 评论中的图片 URL（如有）
  • 评论中的视频 URL（如有）
  • 产品变体信息（如颜色、尺寸等）

重要说明：这些评论数据均为 Amazon 网站上的公开信息，我们仅收集您主动选择采集的评论。

1.1.4 使用数据

  • 采集配置（选择的星级、页数等）
  • 功能使用情况（用于改进产品体验）
  • 错误日志（用于问题诊断）

2. 信息使用方式 / How We Use Information

我们使用收集的信息用于以下目的：

2.1 核心功能

  • 评论采集：根据您的配置采集 Amazon 产品评论
  • 数据存储：将采集的评论数据存储在我们的服务器上，供您后续查看和分析
  • 翻译服务：将评论翻译为您选择的语言
  • 数据分析：对评论进行情感分析、主题提取等智能分析

2.2 服务改进

  • 优化采集算法和性能
  • 修复错误和问题
  • 开发新功能

2.3 用户支持

  • 响应用户请求和问题
  • 提供技术支持

3. 数据存储 / Data Storage

3.1 本地存储

以下信息仅存储在您的浏览器本地，不会上传到服务器：

  • 用户认证令牌（Token）
  • 用户基本信息（邮箱、用户名）
  • Token 过期时间

这些数据存储在 Chrome 的 chrome.storage.local API 中，只有本扩展可以访问。

3.2 服务器存储

以下数据会传输并存储在我们的服务器上：

  • 采集的产品信息
  • 采集的评论数据
  • 翻译后的评论内容
  • 分析结果（情感、主题等）

数据存储在安全的数据库服务器上，采用行业标准的安全措施进行保护。

3.3 数据保留

我们会在您使用服务期间保留您的数据。如果您希望删除数据，可以通过以下方式：

  • 在 Web 控制台中删除特定产品或评论
  • 联系我们的支持团队请求删除账户及相关数据

4. 数据共享 / Data Sharing

4.1 我们不会出售您的数据

我们承诺不会向第三方出售、交易或出租您的个人信息或采集的评论数据。

4.2 有限的数据共享

我们仅在以下情况下可能与第三方共享数据：

  • 服务提供商：与帮助我们运营服务的第三方服务商（如云存储、翻译 API 提供商）共享必要数据
  • 法律要求：当法律要求或为保护我们的权利时
  • 用户同意：在您明确同意的情况下

4.3 第三方服务

本扩展使用以下第三方服务：

  • 翻译服务：使用 AI 翻译 API 进行评论翻译（具体服务商可能变更）
  • 云服务：使用云服务提供商存储数据

这些服务提供商有自己的隐私政策，我们建议您查阅其政策。

5. 数据安全 / Data Security

我们采取以下安全措施保护您的数据：

  • 加密传输：所有数据传输均使用 HTTPS 加密
  • 安全存储：数据存储在安全的服务器环境中
  • 访问控制：仅授权人员可以访问数据
  • 定期备份：定期备份数据以防止数据丢失
  • 安全审计：定期进行安全审计和漏洞扫描

重要提醒：虽然我们采取了合理的安全措施，但没有任何系统是 100% 安全的。请妥善保管您的账户信息，不要与他人分享您的登录凭证。

6. 您的权利 / Your Rights

根据适用的数据保护法律，您享有以下权利：

6.1 访问权

您可以随时访问、查看您账户中存储的所有数据。

6.2 更正权

您可以更正或更新您的个人信息。

6.3 删除权

您可以请求删除您的账户和相关数据。删除后，数据将无法恢复。

6.4 数据导出

您可以请求导出您的数据（如评论数据、分析结果等）。

6.5 撤回同意

您可以随时撤回对数据处理的同意。撤回后，我们可能无法继续为您提供某些服务。

7. Cookie 和本地存储 / Cookies and Local Storage

7.1 Chrome Storage API

本扩展使用 Chrome 的 chrome.storage.local API 在您的浏览器本地存储以下信息：

  • 用户认证令牌
  • 用户基本信息
  • 应用配置和偏好设置

这些数据仅存储在您的设备上，不会与我们的服务器同步（除非您登录账户）。

7.2 第三方 Cookie

本扩展不会在 Amazon 网站上设置任何 Cookie。我们仅读取 Amazon 网站上的公开内容。

8. 儿童隐私 / Children's Privacy

本扩展不面向 13 岁以下的儿童。我们不会故意收集 13 岁以下儿童的个人信息。如果您是家长或监护人，发现您的孩子向我们提供了个人信息，请立即联系我们，我们将删除这些信息。

9. 国际数据传输 / International Data Transfers

您的数据可能会被传输到您所在国家/地区以外的服务器进行处理和存储。我们确保采取适当的安全措施保护您的数据，无论数据存储在何处。

10. 隐私政策变更 / Changes to Privacy Policy

我们可能会不时更新本隐私政策。重大变更时，我们会在扩展中或通过其他方式通知您。继续使用本扩展即表示您接受更新后的隐私政策。

我们会在本页面顶部标注"最后更新日期"，建议您定期查看本政策。

11. 第三方网站 / Third-Party Websites

本扩展会访问 Amazon 网站以采集评论数据。请注意，Amazon 有自己的隐私政策和服务条款。我们不对 Amazon 的隐私做法负责。

本扩展仅读取 Amazon 网站上的公开信息，不会访问您的 Amazon 账户信息、支付信息或其他私人数据。

12. 权限说明 / Permissions Explanation

本扩展请求以下 Chrome 权限，用途如下：

权限 用途
activeTab 访问当前 Amazon 标签页，用于采集评论数据
storage 在浏览器本地存储用户认证信息和配置
scripting 在 Amazon 页面上注入脚本以提取评论数据
tabs 管理标签页，用于后台采集评论
notifications 在采集完成时向您发送通知

重要：我们仅在您主动使用采集功能时才会访问 Amazon 页面。我们不会在后台自动访问或采集数据。
